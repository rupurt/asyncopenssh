## Quick Start

See [Installing Nix and Direnv](https://github.com/mechanical-orchard/monix/wiki/Installing-Nix-and-Direnv) for prerequisites.

Perform initial setup with:

```console
> direnv allow
```

Then explore other options with:

```console
> dev --help
```
